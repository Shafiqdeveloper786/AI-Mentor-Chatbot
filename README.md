🤖 AI Mentor Elite
Quantum Knowledge Core v4.0.2
AI Mentor Elite is a high-performance conversational AI assistant leveraging the Llama 3.3 (70B) model and the ultra-fast processing power of Groq Cloud. It is designed with a futuristic "Cyberpunk" aesthetic, featuring a sleek glassmorphism UI and smooth animations.

🚀 Key Features
Ultra-Fast Inference: Powered by Groq Llama-3.3-70b for near-instantaneous response generation.

Futuristic UI: Immersive animated gradient backgrounds and glassmorphism message containers.

Persona-Driven: Integrated system instructions to maintain a professional "AI Mentor" personality.

Mobile Optimized: Fully responsive design that ensures a seamless experience across all devices.

Auto-Scroll Logic: Enhanced user experience with automated chat-scroll functionality.

🛠️ Tech Stack
Frontend: Streamlit (with Custom CSS Injection)

LLM: Llama 3.3 70B (via Groq API)

Language: Python

Deployment: Streamlit Cloud / Vercel
